# Stellafy - Open source Electron Music Player

Adless, High quality music player that supports SC, Youtube music and soundcloud, all in one.
You can also import Playlists from spotify!

To run, Run `npm install` and to build binary, just build it as you would build an electron app.
after installation, just run the `npm start` to start the application.

You're free to modify it however you like idc
